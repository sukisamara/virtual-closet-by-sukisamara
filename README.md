# ✦ Virtual Closet by Suki Samara

> *dress like the main character*

A full-featured digital wardrobe web app — pure HTML, CSS & JavaScript.  
No installs, no build step. Upload to GitHub and go live immediately.

---

## ✦ How to go live (GitHub Pages)

1. Upload ALL these files to your GitHub repo
2. In your repo, click **Settings**
3. Click **Pages** in the left sidebar
4. Under **Source**, select **Deploy from a branch**
5. Choose **main** branch and **/ (root)** folder
6. Click **Save**

Your site will be live at:
`https://sukisamara.github.io/YOUR-REPO-NAME/`

(Takes ~2 minutes the first time)

## ✦ Features

| Page | Features |
|------|---------|
| **Today** | Daily outfit suggestions · arrow buttons to swap each slot · weather picker · Google Calendar mock sync · log your outfit |
| **My Closet** | Full grid · filter by category / aesthetic / occasion · search · sort · click card to see stats & log wear |
| **Add Item** | 3-step form · photo upload (drag & drop) · 10 categories with subcategories · 16 aesthetics · 12 occasions · colour picker |
| **Calendar** | Monthly calendar · add events · connect Google Calendar · see logged outfits per day |
| **Analytics** | Aesthetic pie chart · cost per wear bar chart · favourite item · category breakdown · most / least worn |
